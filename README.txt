Modern Fullscreen Tetris
------------------------
Files:
- index.html : single-file Tetris game with embedded background image, responsive, ghost, hold, highscore, mobile controls.

How to use:
- Open index.html in a browser (desktop or mobile). No server required.
- Controls:
  Desktop: ← → move, ↑ rotate, ↓ soft drop, Space hard drop, C hold, P pause
  Mobile: on-screen buttons at bottom
